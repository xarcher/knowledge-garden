> Với vai trò là một `Backend Developer`, tôi không chỉ muốn trở thành `thợ gõ phím`, mà còn muốn mở rộng góc nhìn sang lĩnh vực DevOps và Infrastructure Automation.
Series này ra đời như một cuốn nhật ký kỹ thuật - nơi tôi lưu lại quá trình tự học, thử nghiệm và tự động hóa hạ tầng của mình.

### 🧱 Phần 1 — Foundation (Virtualization & Automation)
- Tìm hiểu về Virtualization và ảo hóa
- Tìm hiểu về CI/CD 
- Setup Proxmox
- Tạo VM tự động bằng Terraform + Proxmox Provider
- Quản lý cấu hình bằng Ansible — cài OS, Docker, và công cụ hệ thống

### 🐳 Phần 2 — Containerization & CI/CD Basics
- Docker Deep Dive — Từ image đến Compose và private registry
- Mini Orchestration với Docker Swarm — Hiểu scaling và rolling update
- Build CI/CD Pipeline cơ bản với Jenkins + GitHub

### ☸️ Phần 3 — Kubernetes & Observability
- Triển khai Kubernetes Cluster (k3s/kubeadm) trên lab
- Deploy Redis, Kafka, và ứng dụng thực tế trên Kubernetes
- Quan sát hệ thống với Prometheus, Loki và Grafana

### 🏗️ Phần 4 — Infrastructure as Code & GitOps
- Terraform + Ansible + Helm — “One Command to Build It All”
- GitOps với ArgoCD — Khi CI/CD trở nên tự động hoàn toàn

### 🧠 Bonus
- Lessons Learned — Những sai lầm và bài học từ homelab đầu tiên
- Tổng kết & roadmap tiếp theo — từ homelab tới production
